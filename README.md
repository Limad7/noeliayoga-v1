# NoeliaYoga V1

Primera versión del sitio web de NoeliaYoga, construido con **React + TypeScript + Vite**.

## Stack

- React 19 + TypeScript
- Vite (build tool)
- CSS modular

## Desarrollo

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy

El proyecto incluye un script `deploy.sh` para despliegue.

## Notas

- Versión inicial del sitio, diseño sencillo con Vite puro (sin Tailwind)
- Ver `V2` para la versión rediseñada con Tailwind CSS
